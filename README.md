一款简单小巧便捷的安卓计算器

- 支持四则运算，百分号%和多层括号
- 支持复制粘贴
- 支持任意位置插入数据
- 可保存历史记录，支持单击复制内容及一键清空
- “计算器”界面和“历史记录”界面左右滑屏即可切换
